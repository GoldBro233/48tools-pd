## 技术栈
pug + sass + ECMA7 + react + antd + webpack + nwjs。  

## 文件
* app: 源代码
* ffmpeg: ffmpeg文件存储目录
* output: 视频输出目录

## 功能
* 成员直播录源
* 成员录播下载（开发中）
* B站视频录源（开发中）
* 视频剪切（开发中）
